# Leetcode DSA - Python & JavaScript

A comprehensive collection of LeetCode problem solutions implemented in **Python** and **JavaScript**, focusing on Data Structures and Algorithms (DSA).

## 📋 Overview

This repository contains solutions to various LeetCode problems, organized by difficulty level and topic. Each solution includes:
- Clean, well-commented code
- Time and space complexity analysis
- Multiple approaches when applicable
- Explanations of key concepts

## 🛠️ Languages

- **Python** - For detailed implementations and explanations
- **JavaScript** - For alternative approaches and Node.js solutions

## 📚 Topics Covered

- Arrays & Strings
- Linked Lists
- Stacks & Queues
- Trees & Graphs
- Dynamic Programming
- Sorting & Searching
- Hash Tables
- Greedy Algorithms
- And more...

## 🎯 Difficulty Levels

- Easy
- Medium
- Hard

## 💡 How to Use

1. Navigate to the topic folder of interest
2. Find the problem file (with problem number and name)
3. Review both Python and JavaScript implementations
4. Study the complexity analysis and approach explanation
